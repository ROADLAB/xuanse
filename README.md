# 产品展示页面

这是一个使用 React 开发的产品展示页面，用户可以选择不同的产品类型和颜色来查看对应的产品效果图。

## 功能特点

- 支持选择不同产品类型（平板台面等四种选项）
- 提供9种颜色选择
- 响应式设计，适配各种屏幕尺寸
- 图片等比例缩放显示

## 技术栈

- React
- TypeScript
- CSS3 (Flexbox & Grid)

## 本地开发

1. 克隆项目
```bash
git clone [项目地址]
```

2. 安装依赖
```bash
cd my-react-app
npm install
```

3. 启动开发服务器
```bash
npm start
```

4. 构建生产版本
```bash
npm run build
```
